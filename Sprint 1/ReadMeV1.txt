Code:
- Open a copy of Eclipse that has ADK installed.
- To Run code, either create emulator or use phone with AT LEAST Android 2.3.3 (API 14)
- Log in (actual login not yet implemented, will just redirect)
- The Edit Lists Screen will appear.
- You can choose which list you'd like to view with the spinner at the top of the screen.
- Click "Add Item" to add a new item to the selected list
- Click "Edit List Name" to change the name of the selected list
- Click "Add User" to add a user to the selected list
- Click phone's back button to return to previous screen.
